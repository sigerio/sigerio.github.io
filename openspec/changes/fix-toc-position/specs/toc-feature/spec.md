# 规格变更：文章目录功能

- **capability**: `toc-feature`
- **关联变更**: `fix-toc-position`

## MODIFIED Requirements

### REQ-TOC-001：文章目录位置

**描述**：文章目录应固定在视口右侧，不随页面滚动。

**优先级**：高

**验收标准**：
- 目录使用 `position: fixed` 固定在视口右侧
- 滚动页面时目录位置保持不变
- 目录过长时显示滚动条，可独立滚动

#### Scenario: 目录固定不动
- **GIVEN** 用户在桌面浏览器访问文章页
- **WHEN** 用户滚动页面内容
- **THEN** 目录保持在视口右侧固定位置
- **AND** 目录不随页面内容滚动

#### Scenario: 目录内部滚动
- **GIVEN** 文章目录项较多，超出可视高度
- **WHEN** 用户将鼠标悬停在目录区域
- **THEN** 目录区域显示滚动条
- **AND** 用户可独立滚动目录内容

---

## ADDED Requirements

### REQ-TOC-003：滚动条样式

**描述**：目录滚动条应有明显的视觉样式。

**优先级**：中

**验收标准**：
- 滚动条宽度适中（约 6px）
- 滚动条颜色与主题色协调
- 滚动条圆角美观

#### Scenario: 滚动条可见
- **GIVEN** 目录内容超出可视高度
- **WHEN** 目录区域出现滚动条
- **THEN** 滚动条颜色明显可见
- **AND** 滚动条滑块与轨道有明显区分
