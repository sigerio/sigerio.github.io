## ADDED Requirements

### REQ-CAT-003：分类卡片整体可点击

**描述**：首页分类卡片整体区域应可响应点击事件，点击任意位置均可跳转到对应分类页。

**优先级**：中

**验收标准**：
- 整个卡片区域可点击，而非仅文字部分
- 点击卡片任意位置跳转到对应分类页
- 鼠标悬停时整个卡片有视觉反馈
- 保持现有的悬停上浮效果

#### Scenario: 点击卡片任意位置
- **GIVEN** 用户在首页查看分类卡片
- **WHEN** 用户点击卡片的任意位置（标题、描述、空白区域）
- **THEN** 页面跳转到对应的分类页面

#### Scenario: 卡片悬停效果
- **GIVEN** 用户在首页查看分类卡片
- **WHEN** 鼠标悬停在卡片上
- **THEN** 整个卡片显示悬停效果（上浮）
- **AND** 鼠标指针变为手型

#### Scenario: 键盘无障碍访问
- **GIVEN** 用户使用键盘导航
- **WHEN** 焦点移动到分类卡片
- **THEN** 卡片显示焦点指示
- **AND** 按回车键可跳转到分类页
