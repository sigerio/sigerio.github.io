# 提案：优化目录位置与分类点击体验

## 变更标识
- **change-id**: `improve-toc-and-category-ux`
- **创建日期**: 2025-11-27
- **状态**: 草案

## 背景

用户反馈两个体验问题需要优化：
1. 文章目录位于内容上方，阅读长文章时需要滚动回顶部查看目录
2. 首页分类卡片只有文字部分可点击，不够直观

## 问题摘要

### 问题1：文章目录位置不便 [中等]

**现状**：目录位于文章内容上方，用户滚动后无法快速访问

**期望**：目录固定在页面右侧，滚动时始终可见

**影响**：提升长文章阅读体验

### 问题2：分类卡片点击区域小 [低]

**现状**：`.category-card` 中只有 `<a>` 标签可点击

**期望**：整个卡片区域均可响应点击

**影响**：提升首页交互体验

## 范围

| 范围 | 包含 |
|------|------|
| 布局调整 | TOC 移至页面右侧固定位置 |
| 交互优化 | 分类卡片整体可点击 |

## 成功标准

1. 文章目录固定在页面右侧，滚动时保持可见
2. 首页分类卡片整体可点击，点击任意位置均可跳转
3. 移动端响应式适配正常
